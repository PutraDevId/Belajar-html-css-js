gambar yang bisa di peroses oleh HTML diantaranya JPG,PNG,GIF

atribut yang di gunakan

  • src, berfungsi untuk menentuka source file gambar
  • alt, berfungsu untuk menampilkan tulisan jika gambar tidak di tampilkan
  • Border, berfungsi untuk menentukan garis tepi pada gambar
  • style, berfungsi untuk membuat styling dari gambar ( disarankan menggunakan css external)
