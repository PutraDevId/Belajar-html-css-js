link, berfungsi untuk mengontrol warna link atau tautan agar berbeda dari link lainnya

1.:link, berfungsi untuk mengatur warna link

Contoh:

tautan berwarna hitam

2.:visited, berfungsi untuk mengatur warna link setelah di klik

Contoh:

berubah warna setelah di klik

3.:hover, berfungsi untuk mengatur warna link ketika mouse berada di atas tautan

Contoh:

berubah ketika mouse beradadi sini

4.:active, berfungsi untuk mengatur warna link ketika sedang di klik

Contoh:

berubah ketika sedang di klik
