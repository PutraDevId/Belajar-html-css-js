B.font, berfungsi untuk menentukan jenis font yang di gunakan

1.font-family, berfungsi untuk menentukan jenis font yang digunakan

contoh:

paragraf ini menggunakan font Georgia, Times New Roman, Times, serif

2.font-style, berfungsi untuk menentukan style dari font seperti italic atau oblique. nilai yang digunakan "normal, italic, oblique"

contoh:

paragraf ini menggunakan font style italic

paragraf ini menggunakan font style normal

paragraf ini menggunakan font style oblique

3.font-variant, berfungsi untuk mengubah huruf kecil menjadi besar dari font tetapi ukuran tetap menggunakan huruf kecil. nilai yang di gunakan "small, normal, caps"

contoh:

font pada paragraf ini akan di ubah menjadi huruf besar

font pada paragraf ini akan normal
