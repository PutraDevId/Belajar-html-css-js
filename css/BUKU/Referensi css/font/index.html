B.font, berfungsi untuk menentukan jenis font yang di gunakan

1.font-family, berfungsi untuk menentukan jenis font yang digunakan

contoh:

paragraf ini menggunakan font Georgia, Times New Roman, Times, serif

2.font-style, berfungsi untuk menentukan style dari font seperti italic atau oblique. nilai yang digunakan "normal, italic, oblique"

contoh:

paragraf ini menggunakan font style italic

paragraf ini menggunakan font style normal

paragraf ini menggunakan font style oblique

3.font-variant, berfungsi untuk mengubah huruf kecil menjadi besar dari font tetapi ukuran tetap menggunakan huruf kecil. nilai yang di gunakan "small, normal, caps"

contoh:

font pada paragraf ini akan di ubah menjadi huruf besar

font pada paragraf ini akan tetap normal

4.font-weight, berfungsi untuk meningkatkan atau mengurangi tebal dari font nilai yang dapat di gunakan "normal, bold, bolder, lighter, 100 sampai 900"

contoh:

font pada paragraf ini akan normal

font pada paragraf ini akan bold

font pada paragraf ini akan bolder

font pada paragraf ini akan lighter

font pada paragraf ini akan 900

paragraf ini memiliki ukuran xx-small

paragraf ini memiliki ukuran x-small

paragraf ini memiliki ukuran small

paragraf ini memiliki ukuran medium

paragraf ini memiliki ukuran large

paragraf ini memiliki ukuran xx-large

paragraf ini memiliki ukuran smaller

paragraf ini memiliki ukuran larger

contoh:

text ini menggunakan font size adjust

font-stretch, berfungsi untuk mengatur stratch/rentangan dari font. nilai yang dapat di gunakan "normal, wider, narrower, ultra-condensend, extracondensed, condensed, semi-condensed, semi-expanded, expanded, extra-expanded, ultra-expanded"
